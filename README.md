
# Cara Membuat Bot Telegram OpenAI

## Bahan Yang Perlu di Download
  1. Silakan Install Phyton
  2. Install Git Bash. 
  3. Buat API OpenAI ( https://beta.openai.com/account/api-keys )
  4. Buat bot telegram di @botfather ( dapatkan API )


## Installation

  git clone 
  cd x
